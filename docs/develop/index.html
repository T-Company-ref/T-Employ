1. 요구사항 개요

핵심 화면

신규 지원자 목록 / 지원자 상세 / 인재검색 후보 목록

핵심 자동화

매일 08:00 전일 지원자 요약 메일

핵심 이력

면접탈락/취업성공 등 상태 변경 이력 추적

핵심 보관

지원서 PDF + 공고 스냅샷 보관

2. 신규 지원자 기능 명세

"신규 지원자" 메뉴는 자사 공고에 새로 지원한 사람을 빠르게 확인하고 후속 조치하기 위한 화면입니다.

필수

NA-01 신규 지원자 자동 수집

  • 사람인/잡코리아/커리어 기업 계정으로 공고별 지원자 목록 수집
  • 지원일시, 지원 공고, 기본 연락처를 함께 저장
필수

NA-02 신규 배지 기준

  • 최근 24시간 이내 수집된 지원자는 `NEW` 표기
  • 담당자가 확인 처리하면 신규 목록에서 제외
필수

NA-03 신규 목록 필터

  • 필터: 지원일(일/주), 플랫폼, 공고명, 담당자, 현재 단계
  • 정렬: 최신지원순, 추천순, 면접예정일순
필수

NA-04 빠른 액션

  • 목록에서 바로 "눈여겨봄", "면접검토", "메일발송" 실행
  • 액션은 즉시 이력에 저장

3. 지원자 상세 기능 명세

AD-01 기본 정보

  • 이름, 연락처, 지원일, 지원 플랫폼, 지원 공고명
  • 동일 인물의 과거 지원 이력(중복 지원) 확인

AD-02 문서 확인

  • 지원서 PDF, 이력서 PDF 원본 다운로드/미리보기
  • 문서 수집 실패 시 재수집 버튼 제공

AD-03 상태 이력

  • 지원완료 → 면접진행 → 면접탈락/입사 등 단계 변경
  • 타사 취업 성공 여부(employed_elsewhere) 기록

AD-04 임원 추천

  • 복수 임원이 한 지원자를 추천 가능
  • 추천 코멘트와 추천 시점 이력 저장

4. 인재검색/포지션 제안 기능 명세

필수

TP-01 기업 계정 인재검색 수집

  • 잡코리아 기업 로그인 후 인재검색/포지션 제안 대상 수집
  • 사람인 기업 로그인 후 인재풀 조회 대상 수집
  • 검색 조건, 후보자 프로필 URL, 제안 대상 시점을 저장
필수

TP-02 별도 화면 제공

  • 공고 지원자 화면과 완전히 분리된 전용 목록 사용
  • 지표(건수/전환율)도 별도 계산

6. 대시보드 기능 명세

DB-01 지원자 지표

  • 일별/주별 신규 지원자 수
  • 공고별 지원자 수 및 단계별 전환율

DB-02 관심 후보 지표

  • 눈여겨본 지원자 수
  • 임원 추천 지원자 수

7. 메일 기능 명세

기능설명
ML-01 후보자 개별 메일회사 메일 계정으로 템플릿 발송, 발송결과 저장
ML-02 자동 요약 메일매일 08:00 KST 전일 지원 현황 요약 메일 자동 발송
ML-03 실패 재시도발송 실패 시 재시도 후 운영자 알림

8. 운영 정책

공고지원(Applicants)과 인재검색/포지션제안 후보(Talent Pool)는 데이터/지표/UI를 반드시 분리해 운영합니다.
  • 서로 다른 탭, 서로 다른 집계, 서로 다른 상태 흐름을 사용
  • 지원자 목록/인재검색 후보 목록 간 직접 혼합 정렬 금지

9. 지원자 처리 흐름

1

플랫폼 동기화: 공고/지원자/문서 메타 수집

2

문서 저장: PDF 원본 보관 및 텍스트 추출

3

평가/추천: 채용담당·임원 추천/메모 누적

4

상태관리: 면접진행, 면접탈락, 최종입사, 타사취업 기록

5

후속 액션: 적합 후보 메일 발송 및 이력 저장

6

리포팅: 전일 활동 08:00 자동 요약 발송

주의: 한 명의 후보자가 여러 공고에 지원 가능하므로, 후보자(candidates)와 지원이력(applications)은 분리 모델을 유지해야 합니다.

역할권한

뷰어대시보드 조회 전용

11. 운영 기준

  • 모든 상태 변경은 candidate_status_history append-only 기록
  • 파일 저장소는 해시 중복 제거 + 접근권한 제어
  • 수집/인증/메일 실패는 운영 알림 채널 통합
  • 개인정보 보관 기간 및 파기 정책 별도 문서화

12. 진행 순서

Phase 1 (2~3주): 기반

  • DB 스키마, 스토리지, 공고/지원자 수집 파이프라인
  • 사람인/잡코리아 로그인 수집 MVP

Phase 2 (2주): 운영 기능

  • 대시보드, 상태 이력, 임원 추천 UI
  • 회사 메일 연동 및 템플릿 발송

Phase 3 (1~2주): 자동화 고도화

  • 08:00 전일 요약 리포트 자동 발송
  • 외부 이력서 홍보 수집 탭 및 키워드 관리
  • 공고 스냅샷 조회/운영알림 고도화
